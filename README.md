# data-comparsion
A simple web tool to dynamicly compare application type statuses between JSON exports and Excel datasets.

## Features
- Direct dynamic file uploads for both Excel and JSON datasets.
- Smart column detection to map values automatically.
- Live detail viewing to identify missing records from either system.
- Comprehensive Status Summary to extract duplicate records and sum their totals across two data mediums.
- Grouping mechanisms that abstract generic categories into strict `PBG` or `SLF` master categories.
- Automatic data normalization to combat NULL / None / Empty string discrepancies naturally.
